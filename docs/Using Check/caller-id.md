---
title: Caller ID
---
Here's a trick that gets more stores to pick up: **your checks can call from your own number.**

Verify your number once and every check you run shows the store your caller ID instead of ours. To the store it's a normal local call from a real person's phone, and local calls get answered.

How to set it up:

1. Open your account and start **caller ID verification**.
2. We show you a short code.
3. Your phone rings. Answer and key in the code.
4. Done. Every check you run now dials as you.

You never talk on these calls. Check AI still does all the work; the store just sees a friendlier number.
