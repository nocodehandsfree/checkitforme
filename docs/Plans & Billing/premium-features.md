Every paid plan unlocks the same premium set. It's a membership perk, so pay as you go doesn't include it.

| Feature | What it does |
|---|---|
| **Exact products** | Ask for a specific set on the call. Check several products in one call. |
| **Zone sweeps** | Check every store near you with one tap. |
| **Restock alerts** | Get pinged when a store restocks. |
| **Scheduled checks** | Auto check on a schedule, like shipment days. |
| **Any town** | Search anywhere, past your 20 mile radius. |
| **Store holds** | Ask the store to hold a product for you. |
| **Your voice** | Personalize the voice of your calling agent. |
| **Thrift hunts** | Include thrift stores and hobby shops. |

A few of these are still rolling out to customers, but they're all wired and ready.
