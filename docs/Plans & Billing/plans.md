---
title: Plans
---
Check has four monthly plans. Every plan includes a set number of checks each month. A check is one call to one store.

| Plan | Monthly | Checks a month |
|---|---|---|
| **Family** | $4.99 | 20 |
| **Collector** | $9.99 | 50 |
| **Hunter** | $19.99 | 125 |
| **Operator** | $49.99 | 400 |

Pay for a year up front and it's 17% off. That's about two months free.

Plan checks reset each cycle and don't roll over. Need more mid month? Grab a [pay as you go](doc:pay-as-you-go) bundle. Those never expire.

Every plan also unlocks the full set of [Check+ features](doc:what-check-plus-is).
