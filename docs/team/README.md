# team/ — one folder per agent. Your folder is the ONLY place you start.
| Folder | Agent | Owns |
|---|---|---|
| `devops/` | Check - DevOps | backend, infra, security, deploys, API contract, these docs |
| `website/` | Check - Website | the consumer site (checkit.html) + /pub |
| `admin/` | Check - Admin | the admin dashboard (app.html) + /api |
| `data/` | Check - Data Dev | store rows, importer, chains, store logos |
| `mapping/` | Check - Mapping | phone-tree mapper → locked call recipes |
| `copy/` | Check - Copy | every in-product word (site + admin) |
| `readme/` | Check - ReadMe | public docs on readme.com |
| `design/` | Check - Design | comps, brand, look & feel |
| `qa/` | Check - QA | read-only verification on staging |

**Every folder = `handoff.md`** (stable: who you are, your lane — rarely changes) **+ `checkpoint.md`**
(volatile: current state — update at every "Checkpoint"). Boot = HANDOFF.md + your two files. Nothing else.
