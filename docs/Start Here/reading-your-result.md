Every check comes back as a short verdict. Color carries the meaning, so a glance tells you the story before you read a word.

| | Verdict | What it means |
|---|---|---|
| 🟢 | **In stock** | They've got it. Go grab it before it's gone. |
| 🔴 | **Not in / Sold out** | They don't have it right now. Save the trip. |
| 🟡 | **Couldn't tell** | The answer wasn't clear. Read the convo and you decide. No charge. |
| ⚪ | **No answer** | Nobody picked up, or the line was dead. No charge, try again later. |

Under the verdict you get the actual conversation, the real back-and-forth between Check AI and the store, so you're never taking our word for it. If a store says a shipment landed this morning, you'll see them say it.
