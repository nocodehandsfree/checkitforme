# Pay as you go

No subscription? Buy a **bundle of checks** — the balance never expires, and the price per check drops as you buy more.

| Bundle | Price | Per check |
|---|---|---|
| 10 checks | $9.90 | 99¢ |
| 25 checks | $19.99 | 80¢ |
| 50 checks | $34.99 | 70¢ |
| 75 checks | $47.99 | 64¢ |
| 100 checks | $60.00 | 60¢ |

Pay-as-you-go covers the core "call a store and check stock" flow. **Premium features are subscription-only** — see **[Premium features](doc:premium-features)**.
