# Premium features

Every paid plan unlocks the same set of **premium features** (the "Every plan gets" grid). These are a **membership perk** — **pay-as-you-go customers do not get them.**

| Feature | What it does |
|---|---|
| **Exact products** | Ask for a specific set / product type on the call; check several products in one call |
| **Zone sweeps** | Check every store near you with one tap |
| **Restock alerts** | Get notified when a store restocks |
| **Scheduled checks** | Auto-check on a schedule (e.g. shipment days) |
| **Any town** | Search past your 20-mile radius, anywhere |
| **Store holds** | Ask the store to hold a product for you |
| **Your voice** | Personalize or clone the voice of your calling agent |
| **Thrift hunts** | Include thrift stores & hobby shops |

Each feature can be turned **on or off per plan** in **Admin → God View → Plans** — it's a simple toggle matrix, no code change or deploy needed. Today everything is **on for all paid plans by default**. (A few features aren't launched to customers yet, but they're all wired and ready.)
