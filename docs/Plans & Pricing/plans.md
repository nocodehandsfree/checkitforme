# Plans

Check has **four monthly plans**, low to high. Annual billing is the same plan at **−17%** (roughly two months free). Every plan includes a set number of **checks** each billing cycle (a "check" = one phone call to a store).

| Plan | Monthly | Checks / month |
|---|---|---|
| **Family** | $4.99 | 15 |
| **Collector** | $9.99 | 30 |
| **Hunter** | $19.99 | 100 |
| **Operator** | $49.99 | 300 |

Subscription checks reset each cycle (no rollover). Need more mid-cycle? Add a **[Pay-as-you-go](doc:pay-as-you-go)** bundle — those never expire.

Prices and check counts are owner-editable in **Admin → God View → Plans**, then published to Stripe with one click.
