# Check Brand Style Guide

## Core identity

**Brand name:** Check  
**Approved mark:** Circular green checkmark mark used as the **C** in `Check`.

This pack includes the **brand mark only** as SVG and transparent PNG exports.

## Color palette

### Primary green
- **HEX:** `#4CF286`
- **RGB:** `76, 242, 134`
- **Usage:** Primary brand mark fill, success state, active check UI.

### Highlight green
- **HEX:** `#75F18F`
- **RGB:** `117, 241, 143`
- **Usage:** Optional rim light / subtle glow edge.

### Glow green
- **HEX:** `#19B145`
- **RGB:** `25, 177, 69`
- **Usage:** Neon glow treatments only.

### Background
- **HEX:** `#08090D`
- **RGB:** `8, 9, 13`
- **Usage:** Preferred dark background for hero / UI presentation.

## Brand mark geometry

- Circular green mark with a **check-shaped negative-space cutout**.
- The right arm of the cutout opens through the right edge of the circle.
- Keep the geometry intact.
- Do **not** round the shape further.
- Do **not** change the proportions of the check or the circle opening.

## Preferred usage

### Flat / production version
Use the flat mark for:
- SVG / vector exports
- print
- embroidery
- favicons / app icons
- transparent PNG delivery

### Glow version
Use glow only on dark backgrounds.
Suggested glow stack:

```css
filter:
  drop-shadow(0 0 4px rgba(117, 241, 143, 0.85))
  drop-shadow(0 0 12px rgba(25, 177, 69, 0.60))
  drop-shadow(0 0 28px rgba(25, 177, 69, 0.35));
```

- 2048px

## Clear space

Maintain clear space around the mark equal to at least **0.5×** the mark height.
Preferred clear space: **1×** the mark height.

## Do
- Use `#4CF286` as the base fill.
- Use `#08090D` or transparent background for export.
- Keep the mark proportions locked.
- Use glow only as an effect layer, not as a geometry change.

## Do not
- Change the circle geometry.
- Alter the cutout angle.
- Recolor the mark to another green.
- Put the flat version on a busy background without contrast.
- Add unintended blur or raster artifacts.
