# Set banner art (full key art)

Drop a banner image here and it renders on `/logo-wall/sets` as **muted art + enlarged logo** (the design). 
Name the file by the set **code** (or api id), any of `.jpg/.jpeg/.png/.webp`. Landscape (~16:9) looks best.

Example: Pitch Black → `me05.jpg` · Mega Evolution → `me01.jpg` (or `me1.jpg`).

No file = the set shows a clean **"banner coming soon"** placeholder (nothing is faked).
